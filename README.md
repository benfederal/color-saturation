# Color Saturation

A lightweight Minecraft client-side mod that lets you adjust color saturation, hue, brightness and contrast directly in-game — without touching your monitor settings.

## Features

- **Saturation** — Make colors more vivid or more grey
- **Hue** — Shift the color tone of the entire screen
- **Brightness** — Increase or decrease overall brightness
- **Contrast** — Adjust the contrast of the image

## How it works

The mod applies a lightweight post-processing fragment shader directly to the rendered frame before it is sent to your display. This means near-zero performance impact — no heavy shader packs required.

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/)
2. Install [Fabric API](https://modrinth.com/mod/fabric-api)
3. Install [Cloth Config](https://modrinth.com/mod/cloth-config)
4. Install [Mod Menu](https://modrinth.com/mod/modmenu)
5. Drop the mod `.jar` into your `mods` folder

## Configuration

Open **Mod Menu** → **Color Saturation** or press your assigned keybind to open the config screen.

| Option | Default | Description |
|--------|---------|-------------|
| Hue | 0.0 | Shifts the color tone |
| Saturation | 0.0 | Adjusts color vividness |
| Brightness | 0.0 | Adjusts overall brightness |
| Contrast | 0.0 | Adjusts contrast |
